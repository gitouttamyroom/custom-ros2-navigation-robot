# custom-ros2-navigation-robot
Custom differential drive robot built in ROS2 with SLAM mapping, AMCL localization, and autonomous navigation using the Nav2 stack in a Gazebo simulation environment.

# Custom ROS2 Navigation Robot

This project implements a custom differential drive robot capable of autonomous navigation using ROS2.

## Features
- Custom URDF robot model
- Gazebo simulation
- LiDAR based SLAM mapping
- AMCL localization
- Autonomous navigation using Nav2
- Waypoint based navigation

## Tech Stack
- ROS2 Humble
- Gazebo
- Nav2
- SLAM Toolbox
- RViz
